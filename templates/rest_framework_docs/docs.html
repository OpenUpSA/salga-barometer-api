Salga Barometer API

The South African Local Government Association (SALGA) is an organisation responsible for local government oversight. SALGA have created a web-based portal called Municipal Barometer This portal was developed to provide reliable municipal level data for analysis, re-packing and collection. This API is a supplement to the Municipal Barometer.
